Actions_Tools
=============
* This program only for devices that have Actions ATM7029 processor.

Features:
==========
* Unlocking CPU and GPU frequency(that locked on some roms).
* Fix freezes.
* Fix the color gamut.
* Recovery backup.
* Recovery installer.
* About modules,system.
* CPU and GPU freq, memory speed indexes.
